# Examples

Ready-to-use apps you can deploy on [runit.dev](https://runit.dev) or with `runit deploy`.

| Example | What it does |
|---------|-------------|
| [Invoice Generator](invoice-generator/) | Create invoices with automatic tax calculation. Enter a client name and amount, get a formatted invoice with tax and totals. |
| [Text Analyzer](text-analyzer/) | Analyze any text: count words and sentences, find the most common words, or detect whether the tone is positive or negative. |
| [Unit Converter](unit-converter/) | Convert temperatures (Celsius, Fahrenheit, Kelvin) and distances (miles, kilometers, feet, meters, and more). |

## How to Use

**On the web:** Copy the code from any example's `main.py`, paste it on [runit.dev](https://runit.dev), and hit "Go Live."

**From the terminal:**

```bash
runit deploy examples/invoice-generator --name "Invoice Generator"
```
